# 亿企 → 用友BIP 凭证转换器

把「亿企记账」分录转换为「用友BIP 凭证查询卡片主表（84 列）」导入格式。
所有数据用 SQLite 逐项存储，支持选择 / 模糊搜索 / 编辑，并完整展示「思路过程」转换的每一步中间结果。

## 功能

1. **SQLite 存储**：导入模版的每条分录逐项入库；8 张字典表（用友科目、亿企科目对照、客户、供应商、银行账户、账簿、项目、费用项目）各自建表，每一项都可存储/编辑。
2. **选择 / 模糊搜索**：分录与字典都支持关键字模糊搜索。
3. **思路过程引擎**：解析科目编码/层级 → 亿企科目映射到用友科目 → 判定辅助核算类型 → 在字典中解析出客户/供应商/费用项目/项目/银行账户编码 → 生成会计期间/凭证类型/系统码等凭证头。界面里逐条展示中间步骤。
4. **导出**：写回用友「凭证查询卡片主表」84 列格式（保留原模板 9 行表头）。
5. **PyQt 界面**：导入 → 浏览/搜索/编辑 → 一键转换（可看思路过程）→ 导出。

## 安装

```bash
pip install -r requirements.txt
```

## 运行界面

```bash
python -m app
```

操作顺序：① 导入（先导字典，再导分录） → ② 分录（浏览/搜索/编辑） → ③ 字典（浏览/搜索/编辑） → ④ 配置（账簿/制单人/币种） → ⑤ 转换/导出。

## 命令行（无界面）

```bash
python -m app.cli --dict 思路过程.xlsx --entries 导入模版.xls --out 用友导入.xlsx
```

## 代码结构

| 文件 | 职责 |
| --- | --- |
| `app/db.py` | SQLite 存储层（分录、8 张字典、配置、输出行） |
| `app/importer.py` | 解析 导入模版/思路过程 并入库 |
| `app/engine.py` | 思路过程转换引擎（科目映射、辅助核算解析、凭证头） |
| `app/template_spec.py` | 输出模板 84 列定义与辅助核算列映射 |
| `app/exporter.py` | 写回用友模板（保留 9 行表头） |
| `app/gui.py` | PyQt5 界面 |
| `app/cli.py` | 命令行管线 |
